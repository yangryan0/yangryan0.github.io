Solar System Animation (C++)

After finishing my C++ class in my senior year of high school, I felt that I needed a project that summed up the main concepts we covered in the class. Hence, I decided on building a solar system animation based on the animation concepts we covered at the end of class. The method I used to create the animation is extremely simple. I used the software ImageMagick and created about a thousand images, where each image is a frame.
